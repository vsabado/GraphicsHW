Source Code

This web page contains links to most of the C++ programs used in this course.

  • README.txt - Has instructions on setting up a Linux VM to run OpenGL.
  • src.tar - Contains all sample OpenGL programs below.
  • Makefile - Run "make" to compile all code on Linux or MacOS.

Simple Programs

  • cube.cpp - Demonstrate use of line drawing to display 3D cubes.
  • cube2.cpp - Demonstrate use of polygon drawing to display 3D cubes.
  • cube3.cpp - Demonstrate use of depth buffer when displaying 3D cubes.
  • disk.cpp - Demonstrate color interpolation when displaying a disk.
  • line.cpp - Generate random line sequences using recursive splitting.
  • square.cpp - Demonstrate basic OpenGL commands to display two squares.
  • square2.cpp - Demonstrate color interpolation when displaying two squares.

Keyboard Callbacks

  • building.cpp - Demonstrate x,y rotations of wall model using keyboard callbacks.
  • building2.cpp - Demonstrate x,y rotations of wall model using keyboard callbacks.
  • gasket2d.cpp - Recursive display of 2D Sierpenski gasket using keyboard callbacks.
  • gasket3d.cpp - Recursive display of 3D Sierpenski gasket using keyboard callbacks.
  • office.cpp - Demonstrate x,y rotations of office furniture using keyboard callbacks.
  • office2.cpp - Demonstrate x,y rotations of office furniture using keyboard callbacks

Mouse Callbacks

  • building3.cpp - Demonstrate use of mouse callbacks to draw lines.
  • curve.cpp - Demonstrate use of mouse callbacks to draw curves.
  • office3.cpp - Demonstrate use of mouse callbacks to rotate office furniture models.
  • office4.cpp - Demonstrate use of mouse callbacks to rotate office furniture models.
  • rectangles.cpp - Demonstrate use of mouse callbacks to draw color rectangles.

Idle Callbacks

  • pong.cpp - Demonstrate use of idle callbacks to implement the classic pong game.
  • tree.cpp - Demonstrate use of idle callbacks to generate fractal trees.
  • whack_a_mole.cpp - Demonstrate use of idle callbacks to implement whack a mole game

Object Models

  • object.cpp - Generate and display 3D model of an object of rotation.
  • object2.cpp - Generate 3D model of an object of rotation with surface normals.
  • object3.cpp - Generate and display 3D models of six object of rotation.
  • surface.cpp - Generate fractal surface using recursive rectangle splitting.
  • surface2.cpp - Generate and place random building models on fractal surface.

Phong Shading

  • balls.cpp - Use Phong shading to display bouncing balls within a cubic region.
  • balls2.cpp - Use Phong shading to display exploding balls within a cubic region.
  • balls3.cpp - Use Phong shading to display balls colliding with each other.
  • balls4.cpp - Use Phong shading to display balls attracted towards each other.
  • object4.cpp - Use Phong shading to display six objects of rotation.
  • surface3.cpp - Use Phong shading to display quadratic surface model.
  • surface4.cpp - Use Phong shading to display random wave surface model.
  • shading.cpp - Source file with several functions used by all programs above.

Rendering Algorithms

  • clip.cpp - Demonstrate brute-force line clipping algorithm.
  • line_test.cpp - Compare the speeds of three classic line drawing algorithms.
  • render.cpp - Render random colored rectangles directly into image buffer.
  • render2.cpp - Render random triangles into image buffer using scan-line algorithm.
  • render3.cpp - Render random convex polygons into image buffer using scan-line algorithm.

Texture Mapping

  • mandelbrot.cpp - Generate and display grey-scale Mandelbot set image.
  • mandelbrot2.cpp - Generate and zoom into color Mandelbot set image.
  • texture.cpp - Texture map Mandelbrot set image onto a cube model.
  • texture2.cpp - Texture map wave pattern image onto a cube model.
  • texture3.cpp - Texture map brick photograph onto a cube model.
  • texture4.cpp - Texture map earth from space photograph onto a cube model.
  • texture5.cpp - Texture map earth from space photographs onto objects of rotation.
